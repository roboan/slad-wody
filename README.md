# Ślad Wody

Edukacyjna gra paragrafowa o budowie studni w Czadzie. Projekt jest samodzielną stroną HTML/CSS/JS — nie wymaga instalacji zależności.

## Uruchomienie

Otwórz `index.html` w przeglądarce albo uruchom w tym katalogu prosty serwer lokalny, np. `python -m http.server 8000`, i wejdź na `http://localhost:8000`.

Fabuła i postaci są fikcyjne. Realia techniczne i społeczne opracowano na podstawie materiałów Misji Sercanów, UNICEF i WHO; odnośniki znajdują się w stopce gry.
